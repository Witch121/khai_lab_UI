Lab 1

1. Для чого викоритовується тег div ?
Тег div використовується для групування і стилізації блокових елементів на веб-сторінці. Він не несе семантичного значення і часто використовується для створення контейнерів або блоків для подальшого оформлення CSS.
2. Для чого викоритовується тег section ?
Тег section використовується для виділення розділів або блоків інформації на веб-сторінці. Він допомагає структурувати контент і може використовуватися для групування пов'язаних елементів.
3. Для чого викоритовується тег article ?
Тег article використовується для виділення незалежних, самодостатніх статей або контенту на веб-сторінці. Він допомагає вказати, що цей блок містить інформацію, яка може існувати самостійно.
4. Для чого викоритовується тег p ?
Тег p використовується для визначення абзаців тексту на веб-сторінці.
5. Для чого викоритовується тег a ?
Тег a (англ. anchor) використовується для створення посилань на інші веб-ресурси, сторінки або різні частини тієї ж веб-сторінки.
6. Для чого викоритовується тег nav ?
Тег nav використовується для визначення блоку навігації на веб-сторінці, такого як меню.
7. Для чого викоритовується тег header ?
Тег header використовується для визначення верхньої частини веб-сторінки або блоку, який може містити заголовки, логотипи та інші елементи.
8. Для чого викоритовується тег footer ?
Тег footer використовується для визначення нижньої частини веб-сторінки або блоку, який може містити інформацію про автора, дату публікації та інші елементи.
9. Для чого викоритовується тег form ?
Тег form використовується для створення форм на веб-сторінці для введення користувачем даних.
10. Які типи має тег input?
Тег input має різні типи, такі як text, password, checkbox, radio, email, number, тощо. Кожен тип визначає, який тип введення користувач може використовувати.
11. Як експортувати зображення з фігми?
Для експорту зображень з Figma, можна використовувати опцію "Export" або "Download" у програмі Figma та вибрати потрібний формат та розмір зображення.
12. Як створити посилання на номер телефону або емейл?
Для створення посилання на номер телефону або емейл використовується тег a разом із атрибутом href, який містить відповідний протокол (tel: для телефону, mailto: для електронної пошти).
13. Як додати стилі на веб-сторінку?
Стилі на веб-сторінку можна додавати за допомогою тегу style в розділі HTML-документа або зовнішніми таблицями стилів (CSS).
14. З яких елементів складається базова структура HTML сторінки?
Базова структура HTML-сторінки складається з тегів < html >, < head >, < title >, та < body >.
15. За допомогою яких тегів створюються форми у HTML?
Для створення форм використовуються теги < form >, < input >, < button >, < select >, < textarea >, тощо.
