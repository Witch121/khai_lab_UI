1. Для чого викоритовується тег div ?
Тег div використовується для групування і стилізації блокових елементів на веб-сторінці. Він не несе семантичного значення і часто використовується для створення контейнерів або блоків для подальшого оформлення CSS.
2. Для чого викоритовується тег section ?
Тег section використовується для виділення розділів або блоків інформації на веб-сторінці. Він допомагає структурувати контент і може використовуватися для групування пов'язаних елементів.
3. Для чого викоритовується тег article ?
Тег article використовується для виділення незалежних, самодостатніх статей або контенту на веб-сторінці. Він допомагає вказати, що цей блок містить інформацію, яка може існувати самостійно.
4. Для чого викоритовується тег p ?
Тег p використовується для визначення абзаців тексту на веб-сторінці.
5. Для чого викоритовується тег a ?
Тег a (англ. anchor) використовується для створення посилань на інші веб-ресурси, сторінки або різні частини тієї ж веб-сторінки.
6. Для чого викоритовується тег nav ?
Тег nav використовується для визначення блоку навігації на веб-сторінці, такого як меню.
7. Для чого викоритовується тег header ?
Тег header використовується для визначення верхньої частини веб-сторінки або блоку, який може містити заголовки, логотипи та інші елементи.
8. Для чого викоритовується тег footer ?
Тег footer використовується для визначення нижньої частини веб-сторінки або блоку, який може містити інформацію про автора, дату публікації та інші елементи.
9. Для чого викоритовується тег form ?
Тег form використовується для створення форм на веб-сторінці для введення користувачем даних.
10. Які типи має тег input?
Тег input має різні типи, такі як text, password, checkbox, radio, email, number, тощо. Кожен тип визначає, який тип введення користувач може використовувати.
11. Як експортувати зображення з фігми?
Для експорту зображень з Figma, можна використовувати опцію "Export" або "Download" у програмі Figma та вибрати потрібний формат та розмір зображення.
12. Як створити посилання на номер телефону або емейл?
Для створення посилання на номер телефону або емейл використовується тег a разом із атрибутом href, який містить відповідний протокол (tel: для телефону, mailto: для електронної пошти).
13. Як додати стилі на веб-сторінку?
Стилі на веб-сторінку можна додавати за допомогою тегу style в розділі < head > HTML-документа або зовнішніми таблицями стилів (CSS).
14. З яких елементів складається базова структура HTML сторінки?
Базова структура HTML-сторінки складається з тегів < html >, < head >, < title >, та < body >.
15. За допомогою яких тегів створюються форми у HTML?
Для створення форм використовуються теги < form >, < input >, < button >, < select >, < textarea >, тощо.

1. Що таке reset та normalize підходи та чим вони відрізняються?
Якщо використовувати reset CSS, то всі стилі будуть встановлені відповідно до певних значень за замовчуванням. У випадку з normalize CSS, стилі будуть нормалізовані для забезпечення більш консистентного відображення на різних браузерах, не скидаючи всі стилі до одного базового значення.
2. Для чого використовується мова CSS?
Мова CSS (Cascading Style Sheets) використовується для опису зовнішнього вигляду веб-сторінок, написаних мовою HTML. Основні цілі: -CSS дозволяє розділити опис структури веб-сторінки (HTML) від її візуального вигляду. - CSS дозволяє визначати різні аспекти вигляду елементів HTML, такі як шрифти, кольори, розміри, відступи, рамки, фони та інші стилістичні властивості. - CSS дозволяє створювати респонсивні веб-сайти, які змінюють свій вигляд в залежності від розміру екрану користувача. -CSS дозволяє використовувати одні й ті ж стилі для різних браузерів, забезпечуючи при цьому близьку консистентність у відображенні веб-сторінок. - CSS дозволяє створювати анімацію та переходи, що покращують користувацький досвід та можуть зробити веб-сайт більш привабливим та інтерактивним.
3. Для чого використовується властивість justify-content?
Властивість justify-content використовується в CSS для контролю вирівнювання по головній вісі контейнера flexbox або grid макету. Головна вісь - це вісь, яка використовується для вирівнювання flexbox (зазвичай горизонтальна для рядків і вертикальна для колонок). justify-content має кілька значень: flex-start, flex-end, center, space-between, space-around.
4. Що таке CSS-селектор?
CSS-селектор - це паттерн, який використовується для вибору елементів веб-сторінки, до яких будуть застосовані стилі. Селектори CSS дозволяють точно визначити, які елементи на сторінці повинні бути стилізовані. Основні типи CSS-селекторів: -elements, -ID, -class, -pseudo-class, -pseudo-element, -combined, -attribute
5. Як задати тінь для елемента?
Тінь елемента можна задати використовуючи box-shadow, параметрами якого є горизонтальне розміщення, вертикальне розміщення, розмір розподілення, розміо розширення(опційно) та колір (опційно, але все ж бажано). Напр: .shadow { box-shadow: 10px 10px 5px #888888 };

.send_btn { grid-area: 4/ 2/ 5/ 3; border-radius: 0.375rem; background-color: #42446E; color: whitesmoke; max-width: fit-content; margin-right: 2%; padding-top: 2%; padding-bottom: 2%; padding-left: 10%; padding-right: 10%; display: flex; justify-self: end; } В даному випадку всі елементи з класом "send_btn" матимуть встановлені значення
7. За що відповідає css стиль flex?
CSS властивість flex використовується для встановлення розташування та поведінки елемента в контексті flexbox. flexbox дозволяють елементам вирівнюватися гнучко відносно один одного, що полегшує розміщення елементів на сторінці та створює адаптивний дизайн. Властивість flex має три значення: flex-grow, flex-shrink та flex-basis.
8. За що відповідає css стиль flex-wrap?
CSS властивість flex-wrap використовується для визначення, чи дозволено flex container переносити свої елементи на наступний рядок або колонку в разі нехватки місця в поточному рядку або колонці. Властивість flex-wrap може приймати значення: -wrap, -nowrap and -wrap-reverse
9. Для чого використовується @font-face?
@font-face - це CSS правило, яке дозволяє підключати зовнішні шрифти до веб-сторінок під час розробки. Використання @font-face дозволяє використовувати будь-який шрифт на веб-сторінці, незалежно від того, чи є він встановлений на пристрої користувача.
10. Для чого використовується властивість font-family?
Властивість font-family в CSS використовується для визначення шрифту тексту веб-сторінки. Вона дозволяє вказати один або кілька шрифтів для використання в тексті, і браузер буде вибирати перший доступний шрифт зі списку, який відображатиметься на веб-сторінці.
11. Для чого використовується властивість font-weight?
Властивість font-weight в CSS використовується для визначення жирності тексту, який буде використовуватися для відображення тексту на веб-сторінці. Вона дозволяє контролювати товщину тексту, що дозволяє створювати різноманітністилі тексту на сторінці.
12. За що відповідає css стиль align-items?
CSS властивість align-items використовується для вирівнювання дочірніх елементів flex container вздовж вторинної (перпендикулярної до головної)осі контейнера. Основні значення властивості: -stretch, -baseline, -center, flex-start, -flex-end.

1. За що відповідає стиль transition
Стиль transition в CSS використовується для плавних переходів між двома або більше станами елемента. Він дозволяє анімувати зміни властивостей елемента, таких як колір, розмір, положення тощо, з плавним переходом від одного стану до іншого.
2. Як задати напівпрозорий колір за допомогою css?
Щоб задати напівпрозорий колір за допомогою CSS, можна використати значення RGBA для кольору. Наприклад: background-color: rgba(255, 0, 0, 0.5); задасть червоний колір з напівпрозорістю 50%.
3. Яким чином можна задати стилі для мобільних пристроїв?
Стилі для мобільних пристроїв можна задати за допомогою медіа-запитів (media queries). Наприклад: @media only screen and (max-width: 600px) { /* стилі для мобільних пристроїв */ width: 48%; display: flex; align-items: center; font-size: 16px; }
4. За що відповідає css стиль opacity?
Стиль opacity в CSS використовується для встановлення прозорості елемента. Значення властивості може варіюватися від 0 (повністю прозорий) до 1 (непрозорий).
5. За що відповідає стиль transform?
Стиль transform в CSS використовується для зміни форми, розміру та положення елемента. Він дозволяє виконувати різні трансформації, такі як переміщення, обертання, масштабування тощо.
6. Як підключити js на сторінку?
Щоб підключити JavaScript на сторінку, використовуйте тег "script" у тілі HTML-документу або зовнішній файл JavaScript за допомогою атрибута src.
7. Для чого використовується мета тег meta name="viewport"?
Мета-тег meta name="viewport" використовується для налаштування параметрів відображення сторінки на мобільних пристроях, таких як ширина, масштабування, ініціальне збільшення тощо.
8. Що робить наступна конструкція flex: 1 0 0%;?
Конструкція flex: 1 0 0%; встановлює для гнучкого елемента значення для flex-grow, flex-shrink та flex-basis. У цьому випадку, елемент буде розтягуватися, але не буде зменшуватися, і його базовий розмір буде 0%.
9. Що робить наступний селектор .inner > *?
Селектор .inner > * вибирає всіх прямих дочірніх елементів .inner.
10. Чи відрзняється .header.is-active від .header .is-active?
.header.is-active вибирає елемент з класом is-active, який є дочірнім елементом .header. .header .is-active вибирає елемент з класом is-active, який є вкладеним у елемент .header.
11. Що робить наступна конструкція .form > *:not(:last-child)?
Конструкція .form > *:not(:last-child) вибирає всі дочірні елементи .form, крім останнього елемента.
12. Для чого використовується властивість order для flex елементів?
Властивість order для гнучких елементів flexbox визначає порядок їх розміщення в контейнері. Елементи з меншим значенням order поміщаються перед елементами з більшим значенням order. За замовчуванням, всі елементи мають значення order: 0.

Як працюює метод map?
Метод map(): Цей метод створює новий масив, викликаючи функцію зворотного виклику для кожного елемента в масиві і повертаючи результати цих викликів у новий масив. Наприклад: const numbers = [1, 2, 3, 4, 5]; const doubled = numbers.map(num => num * 2); // doubled = [2, 4, 6, 8, 10]
Як працюює метод join?
Метод join(): Цей метод об'єднує всі елементи масиву в рядок, розділяючи їх заданим роздільником і повертаючи результуючий рядок. Наприклад: const fruits = ["Apple", "Banana", "Orange"]; const result = fruits.join(", "); // result = "Apple, Banana, Orange"
Як працюює метод filter?
Метод filter(): Цей метод створює новий масив, який містить всі елементи вихідного масиву, для яких функція зворотного виклику повертає true. Наприклад: const numbers = [1, 2, 3, 4, 5]; const evenNumbers = numbers.filter(num => num % 2 === 0); // evenNumbers = [2, 4]
Як працюює метод includes?
Метод includes(): Цей метод перевіряє, чи містить масив певний елемент, і повертає true, якщо так, або false, якщо ні. Наприклад: const numbers = [1, 2, 3, 4, 5]; const hasThree = numbers.includes(3); // hasThree = true
Як працюює метод addEventListener?
Метод addEventListener(): Цей метод додає обробник подій до вказаного елемента, який буде викликаний при виникненні певної події. Наприклад: const button = document.querySelector("button"); button.addEventListener("click", () => { console.log("Button clicked!"); });
Як працюює цикл for...in?
Цикл for...in: Цикл for...in використовується для перебору властивостей об'єкта. Він перебирає всі власні властивості об'єкта, а також властивості, що успадковані від прототипу об'єкта. Наприклад: const obj = { a: 1, b: 2, c: 3 }; for (let key in obj) { console.log(key, obj[key]); } // Output: // a 1 // b 2 // c 3
За допомогою якого методу можна задать HTML код елементу?
Для задання HTML-коду елементу можна використовувати властивість innerHTML, наприклад: const element = document.getElementById("example"); element.innerHTML = "< h1 >Hello, world!";
Як працює метод querySelector?
Метод querySelector(): Цей метод шукає перший елемент у документі, який відповідає вказаному селектору CSS. Він повертає перший знайдений елемент або null, якщо елемент не знайдено.
